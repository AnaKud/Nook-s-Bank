# Nook's Bank
Focus Start graduating project
App Description: iOS application for reading News about events in Animal Crossing and Bank App for your Bells Acoount.

Project type: Pet

Tech Stack: MVP, FireBase(Auth&DB), API, CoreData, SnapKit
